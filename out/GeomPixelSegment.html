Class: GeomPixelSegment

GeomPixelSegment(p, v)

An line from one point to a second point Represented by the parametric form of the equation for a line, p + t*v t is constrained to be only between 0 and 1, where 0 gives p and 1 gives the endpoint. v magnitude gives the end point. This version uses GeomPixel classes for p and v.

Name Type Description
p GeomPixelPoint Origination point for the ray.
v GeomPixelVector Direction vector of the ray.
