Class: GeomSegment

GeomSegment(p, v)

An line from one point to a second point Represented by the parametric form of the equation for a line, p + t*v t is constrained to be only between 0 and 1, where 0 gives p and 1 gives the endpoint. v magnitude gives the end point.

point()

Get a point on the line t = 0 is this.p. Each increment of t by 1 is equal to adding the line vector magnitude to the line point. So if this.p = {0, 0, 0} and this.v = {10, 20, -10}, this.point(2) returns {20, 40, -20}. For rays, only positive t are permitted. Negative t will be changed to absolute value with a warning
